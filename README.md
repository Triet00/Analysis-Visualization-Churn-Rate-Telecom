🚀 Project Overview
Telecom companies face high churn rates, and retaining customers is often more cost‑effective than acquiring new ones.
In this project, I:
- Cleaned and prepared the dataset
- Performed EDA to uncover trends and correlations
- Built a Power BI dashboard to visualize churn drivers
- Highlighted key business insights and recommendations
The dashboard allows stakeholders to explore churn by demographics, contract type, service usage, monthly charges, and more.

🧠 Key Questions Explored
- Which customer groups churn the most
- How features such as contract type, age groups, and location influence churn
- Whether add‑on service affect retention

🛠️ Tools & Technologies
- Power BI – Data modeling, DAX, and dashboard creation
- Python (optional) – Initial EDA and data cleaning
- Excel / CSV – Raw dataset

🔍 Insights & Findings
- Customers on month‑to‑month contracts churn significantly more
- Higher monthly charges correlate with higher churn
- Customers mostly churn due to competitor-related reason.
- Customers who are not in a group plan are more likely to churn.
